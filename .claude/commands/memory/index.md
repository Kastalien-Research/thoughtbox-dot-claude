# Memory Commands

Commands for interacting with the memory system - searching, adding, and managing learnings.

## Available Commands

### [start](./start.md)

Quick start guide for the memory system. Get oriented and see what's available.

### [search](./search.md)

Search the memory system for patterns and learnings.

### [add-quick](./add-quick.md)

Add a learning to the memory system in one command.

## Use Cases

### Starting Work

- Check current focus and active context
- See memory system statistics
- Find relevant patterns for your task

### During Work

- Search for patterns when stuck
- Find solutions to similar past problems
- Look up testing conventions

### After Work

- Capture new learnings
- Document tricky solutions
- Update active context

## Example Usage

```bash
# Quick start orientation
/memory:start

# Search for patterns
/memory:search "authentication"

# Add a learning quickly
/memory:add-quick
```
